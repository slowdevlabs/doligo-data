돌리GO 개인정보 처리방침

시행일: 2026년 5월 12일

돌리GO(이하 “앱”)는 사용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보 처리방침은 앱이 어떤 정보를 처리하는지, 사용자의 정보가 어떻게 보호되는지 설명합니다.

현재 돌리GO는 회원가입 없이 사용할 수 있으며, 사용자의 여행 결과와 룰렛 기록은 기기 안에만 저장됩니다.

1. 수집하는 개인정보

앱은 현재 이름, 이메일 주소, 전화번호, 계정 정보 등 사용자를 직접 식별할 수 있는 개인정보를 수집하지 않습니다.

앱 내에서 생성되는 다음 정보는 사용자의 기기 안에 로컬로 저장될 수 있습니다.

  • 도착지 룰렛 결과
  • 음식 룰렛 결과
  • 오늘의 돌리GO 진행 상태
  • 앱 사용 중 생성되는 룰렛 기록

위 정보는 앱 기능 제공을 위해 사용자의 기기 안에 저장되며, 개발자 서버로 전송되지 않습니다.

2. 원격 데이터 사용

앱은 도시, 음식 등 룰렛 후보 데이터를 최신 상태로 유지하기 위해 원격 JSON 데이터를 불러올 수 있습니다. 이 과정에서 앱은 후보 데이터 파일을 다운로드할 수 있으나, 사용자의 룰렛 결과나 개인 기록을 서버로 전송하지 않습니다.

3. 지도 앱 연결

사용자가 “진짜 간다” 또는 “먹으러 간다” 기능을 사용할 경우, 네이버지도, 카카오맵, T맵, Google Maps, Apple Maps 등 외부 지도 앱으로 이동할 수 있습니다.

외부 지도 앱을 사용할 때에는 해당 지도 서비스의 개인정보 처리방침이 적용됩니다. 돌리GO는 외부 지도 앱에서 처리되는 위치 정보나 검색 기록을 수집하거나 저장하지 않습니다.

4. 공유 기능

앱은 사용자가 선택한 룰렛 결과를 이미지 또는 텍스트 형태로 공유할 수 있는 기능을 제공할 수 있습니다. 공유는 사용자가 직접 선택한 경우에만 이루어지며, 앱이 자동으로 결과를 외부에 전송하지 않습니다.

5. 문의하기

사용자가 앱 내 “문의하기” 기능을 통해 이메일을 보내는 경우, 사용자가 직접 작성한 이메일 내용과 발신자 이메일 주소가 문의 처리를 위해 사용될 수 있습니다. 문의 내용은 답변 및 문제 해결 목적으로만 사용됩니다.

6. 제3자 제공 및 광고

앱은 현재 사용자의 개인정보를 제3자에게 판매하거나 제공하지 않습니다. 또한 현재 별도의 광고 SDK 또는 분석 SDK를 사용하지 않는 것을 전제로 합니다.

향후 광고, 분석 도구, 유료 기능 등이 추가될 경우 본 개인정보 처리방침을 업데이트하고 필요한 경우 앱 내 또는 앱스토어 정보를 통해 안내합니다.

7. 데이터 보관 및 삭제

앱의 룰렛 결과와 진행 상태는 사용자의 기기 안에 저장됩니다. 사용자가 앱을 삭제하면 기기에 저장된 앱 데이터도 함께 삭제될 수 있습니다.

앱 내에 기록 삭제 기능이 제공되는 경우, 사용자는 해당 기능을 통해 로컬 기록을 삭제할 수 있습니다.

8. 아동의 개인정보

앱은 고의로 아동의 개인정보를 수집하지 않습니다. 앱은 회원가입이나 개인 식별 정보를 요구하지 않는 방식으로 제공됩니다.

9. 개인정보 처리방침의 변경

본 개인정보 처리방침은 앱 기능 변경, 법령 변경, 서비스 운영 정책 변경에 따라 수정될 수 있습니다. 변경 시 본 페이지의 시행일을 갱신합니다.
